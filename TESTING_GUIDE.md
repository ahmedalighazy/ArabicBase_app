# دليل الاختبار - التنقل من الوحدات لتفاصيل الوحدة

## الخطوات للاختبار:

### 1. تشغيل التطبيق
```bash
flutter run
```

### 2. التنقل للوحدات
- انتظر انتهاء Splash Screen (4 ثواني)
- اضغط "تخطي" أو "هيا نبدأ" في Onboarding
- يجب أن تفتح صفحة الوحدات (Lessons Page)

### 3. اختبار التنقل لتفاصيل الوحدة
- اضغط على أي كارد وحدة (مثلاً: الوحدة الأولى: الفَتْحَة)
- يجب أن تفتح صفحة تفاصيل الوحدة
- يجب أن تشوف:
  - ✅ Header مع زر رجوع وعنوان الوحدة
  - ✅ شريط التقدم العام
  - ✅ كارد الدروس (أيقونة + "الدروس" + "درس واحد في هذه الوحدة")
  - ✅ كارد التدريبات (أيقونة + "التدريبات" + "X تدريبات في هذه الوحدة")

### 4. التحقق من الأيقونات
- تأكد إن أيقونة الدروس ظاهرة صح
- تأكد إن أيقونة التدريبات ظاهرة صح
- لو الأيقونات مش ظاهرة، اعمل Hot Restart (Shift+R)

### 5. اختبار زر الرجوع
- اضغط على زر السهم في الأعلى
- يجب أن ترجع لصفحة الوحدات

## الكود المسؤول عن التنقل:

### في lessons_page.dart:
```dart
LessonCard(
  lesson: lesson,
  onTap: () {
    Navigator.push(
      context,
      MaterialPageRoute(
        builder: (context) => LessonDetailsPage(lesson: lesson),
      ),
    );
  },
)
```

### في lesson_details_page.dart:
```dart
IconButton(
  onPressed: () => Navigator.pop(context),
  icon: Icon(
    Icons.arrow_forward,
    color: AppColors.primary,
  ),
)
```

## المشاكل المحتملة والحلول:

### المشكلة: الأيقونات مش ظاهرة
**الحل:**
1. تأكد إن الملفات موجودة:
   - `assets/images/svg/icons/icon_lesson.svg`
   - `assets/images/svg/icons/icon_trainng.svg`
2. اعمل Hot Restart (Shift+R) مش Hot Reload

### المشكلة: التطبيق بيكراش
**الحل:**
1. شوف الـ Console للأخطاء
2. تأكد إن كل الـ imports موجودة
3. اعمل `flutter clean` و `flutter pub get`

### المشكلة: الصفحة فاضية
**الحل:**
1. تأكد إن البيانات التجريبية موجودة في `_getDemoLessons()`
2. تأكد إن الـ ListView.builder شغال صح
